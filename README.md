
# Basic Discord Web Panel

Basit bir şekilde web panel üzerinden komut eklemenizi sağlayan öğretici bir bot.



## Kurulum

config.json dosyasını düzenleyin.

```bash
  config.json
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  node .
```
